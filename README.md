### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/byronmccormick/shopping-cart.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Start the App:
   ```sh
   npm start
   ```
Open the Expo Go app on your mobile device and scan the QR code from your terminal or browser.
